**对如下表进行行列转换**
![image.png](https://upload-images.jianshu.io/upload_images/21580557-7b38264f7265002d.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)

<br>
**执行**
```sql
select sid,(case project when "math" then score else 0 end) as math,(case project when "english" then score else 0 end) as english from student group by sid;
```
